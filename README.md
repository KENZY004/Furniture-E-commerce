# Nordform E-Commerce

A modern, responsive e-commerce website for furniture shopping built with HTML, CSS, and JavaScript.

## Features

- **User Authentication**: Secure login and registration system with password strength validation
- **Product Catalog**: Browse furniture by categories (Soft Seating, Seating, Tables, Storage, Lighting, Decor)
- **Shopping Cart**: Add products to cart with real-time updates
- **Search Functionality**: Quick product search with live results
- **Checkout System**: Complete checkout flow with order summary
- **Responsive Design**: Mobile-friendly interface that works on all devices
- **Protected Routes**: Authentication-required pages for shopping features

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- LocalStorage for data persistence

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/KENZY004/Furniture-E-commerce.git
   ```

2. Open `home.html` in your web browser to start exploring the website.

3. To access shopping features, create an account via the Sign Up page.

## Features in Detail

### Authentication System
- User registration with validation
- Secure login with "Remember Me" option
- Password strength indicator
- Session management using localStorage/sessionStorage

### Shopping Experience
- Product browsing by category
- Add to cart functionality
- Cart management (add, remove, update quantities)
- Real-time cart total calculation
- Checkout with shipping and payment details

### User Interface
- Clean, modern Scandinavian-inspired design
- Smooth animations and transitions
- Responsive navigation
- Search with autocomplete
- User account dropdown menu
